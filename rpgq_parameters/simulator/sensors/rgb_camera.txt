# parameter file for rgb camera system
#
# index : type: value: description
# type: 0 boolean
#       1 integer
#       2 unsigned
#       3 double
#       4 double3D
0: 3: 30.0: update frequency [Hz]
1: 1: 3: rgb channels [N]
2: 1: 1024: camera width [pixels]
3: 1: 768: camera height [pixels]
4: 3: 70.0: camera field of view
5: 3: 0.20: camera depth scale
